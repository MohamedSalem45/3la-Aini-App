# إعداد أيقونة التطبيق

## الخطوات المطلوبة:

### 1️⃣ حفظ الصورة

قم بحفظ صورة الأيقونة المرفقة في:

```
assets/images/app_icon.png
```

**المتطلبات:**

- حجم الصورة: 1024x1024 بكسل أو أكثر
- صيغة: PNG
- الصورة يجب أن تحتوي على خلفية (شفافة أو ملونة)

### 2️⃣ إنشاء الأيقونات

بعد حفظ الصورة، شغّل الأمر:

```bash
flutter pub run flutter_launcher_icons
```

### 3️⃣ إعادة تشغيل التطبيق

```bash
flutter clean
flutter pub get
flutter run
```

---

## ماذا يحدث؟

الأداة `flutter_launcher_icons` ستقوم بـ:

- ✅ إنشاء أيقونات بأحجام مختلفة لـ Android
- ✅ إنشاء أيقونات لـ iOS
- ✅ إنشاء أيقونات للويب
- ✅ تحديث ملفات المشروع تلقائياً

---

## ملاحظات:

- الصورة الأصلية يجب أن تكون مربعة (1:1 aspect ratio)
- يفضل استخدام صور عالية الجودة
- الخلفية يمكن أن تكون شفافة أو بلون متين
